Cài đặt lần đầu
Nhập Google OAuth Client ID của bạn để bắt đầu. Bạn cần tạo credentials tại Google Cloud Console.
Google OAuth Client ID
1
Vào console.cloud.google.com → Tạo project mới hoặc chọn project có sẵn
2
Vào APIs & Services → Library → Bật Google Drive API
3
Vào APIs & Services → OAuth consent screen → Chọn External → Điền tên app → Thêm scopes: drive.file, userinfo.email, userinfo.profile
4
Vào APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID → Chọn Web application → Thêm Authorised JavaScript origins: địa chỉ bạn mở file này (ví dụ http://localhost:8000)
5
Copy Client ID dán vào ô trên. Lưu ý: Cả 2 máy phải đăng nhập cùng 1 tài khoản Google
